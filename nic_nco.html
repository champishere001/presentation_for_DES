🏢 NIC Code – परिचय

📝 Full Form: National Industrial Classification

ये कंपनी/बिजनेस/फैक्ट्री/दुकान किस तरह की है, बताता है

🤔 आसान भाषा में

"आपका बिजनेस क्या बनाता/बेचता/सर्विस देता है?"

📌 Key Details

• Latest: NIC 2008
• Code Length: 2-5 digits
• जितना detailed, उतना लंबा code

📊 NIC Code के उदाहरण
उदाहरण NIC Code सेक्टर
🍞 रोटी/बिस्किट बनाने वाली कंपनी 1071 Manufacturing (Food)
👔 कपड़े की दुकान 4771 Retail Trade
📱 मोबाइल रिपेयर सर्विस 9521 Repair Services
💻 सॉफ्टवेयर डेवलपमेंट 6201 IT Services
👷 NCO Code – परिचय

📝 Full Form: National Classification of Occupations

ये व्यक्ति/कर्मचारी क्या काम करता है, बताता है

🤔 आसान भाषा में

"आपकी जॉब/पद/occupation क्या है?"

📌 Key Details

• Latest: NCO 2015
• Code Length: 8 digits
• Aligned with ISCO-08 (International)

📊 NCO Code के उदाहरण
जॉब NCO Code Skill Level
🧵 सिलाई मशीन ऑपरेटर 8153.0111 Level 2
📒 अकाउंटेंट 3313.0100 Level 3
🏭 फैक्ट्री मैनेजर 1120.0100 Level 4
🥖 बेकर (रोटी बनाने वाला) 7512.0100 Level 2
⚖️ NIC vs NCO – Side-by-Side Comparison
बात NIC Code 🔵 NCO Code 🟢
क्या classify करता है कंपनी / बिजनेस / उद्योग व्यक्ति / कर्मचारी / occupation
सवाल आपका बिजनेस क्या करता है? आप क्या काम करते हैं?
Code length 2-5 digits (NIC 2008) 8 digits (NCO 2015)
मुख्य यूज Udyam/MSME, GST, Company Reg, Census Job portals, Skill India, Employment Exchange
उदाहरण Bread factory → NIC 1071 Baker in factory → NCO 7512

🌐 कहाँ-कहाँ इस्तेमाल होते हैं?

🔵 NIC Code Uses

  • Udyam Registration
  • GST Registration
  • Bank Loan Applications
  • MSME Benefits
  • Census & Statistics

🟢 NCO Code Uses

  • NCS Portal (National Career Service)
  • Skill Development Schemes
  • Employment Exchange
  • Job Applications
  • PMKVY (Pradhan Mantri Kaushal Vikas Yojana)

🟡 दोनों का Common Use

Labour Statistics, Government Schemes, Economic Planning, Policy Making

🔍 सही Code कैसे ढूंढें?
1

NIC Code के लिए

mospi.gov.in → NIC 2008 search tool पर जाएं

2

NCO Code के लिए

ncs.gov.in या DGE website → NCO 2015 search करें

3

Search Tip

Keywords डालें (जैसे: "bread manufacturing", "accountant") → closest match चुनें

💡 Pro Tip

Exact match न मिले तो सबसे closest category चुनें। जरूरी नहीं कि हर छोटे काम का अलग code हो!

📝 Summary / Key Takeaways
🎯

NIC = Business Activity (कंपनी क्या करती है)

➕

NCO = Job Role (व्यक्ति क्या काम करता है)

दोनों मिलकर देश की economy और employment को track करते हैं

सही code से schemes और benefits आसानी से मिलते हैं!
